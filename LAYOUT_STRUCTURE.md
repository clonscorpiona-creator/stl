# 🏗️ Структура макета STL Platform

## Основные блоки страницы

### 1. **Header** (Шапка)
- **Файл:** `app/components/Header.tsx`
- **Стили:** `app/components/Header.module.css`
- **Описание:** Верхняя панель с логотипом и навигацией
- **Элементы:** Логотип, меню навигации, кнопки входа/регистрации

### 2. **MainSidebar** (Боковая панель)
- **Файл:** `app/components/MainSidebar.tsx`
- **Стили:** `app/components/MainSidebar.module.css`
- **Описание:** Боковое меню с основными разделами
- **Элементы:** Список ссылок на разделы сайта

### 3. **Hero Section** (Главный баннер)
- **Класс CSS:** `.hero`
- **Описание:** Первый экран с главным заголовком и призывом к действию
- **Элементы:**
  - `.heroContent` - контейнер контента
  - `.heroLabel` - метка над заголовком
  - `.heroTitle` - главный заголовок
  - `.heroSubtitle` - подзаголовок
  - `.heroButtons` - кнопки действий

### 4. **Grid Section** (Сетка специализаций)
- **Класс CSS:** `.gridSection`
- **Описание:** Сетка карточек с направлениями творчества
- **Элементы:**
  - `.gridContainer` - контейнер сетки
  - `.gridCard` - карточка специализации
  - `.gridCardImage` - изображение карточки
  - `.gridCardOverlay` - оверлей с текстом
  - `.gridCardTitle` - заголовок карточки
  - `.gridCardSubtitle` - подзаголовок карточки

### 5. **About Section** (О сообществе)
- **Класс CSS:** `.aboutSection`
- **Описание:** Блок с информацией о сообществе
- **Элементы:**
  - `.aboutContainer` - контейнер секции
  - `.aboutImageWrapper` - обёртка изображения
  - `.aboutImage` - изображение
  - `.aboutContent` - текстовый контент
  - `.aboutTitle` - заголовок
  - `.aboutText` - описание
  - `.aboutStats` - звёзды рейтинга
  - `.aboutFooter` - подпись

### 6. **Info Section** (Контакты и статистика)
- **Класс CSS:** `.infoSection`
- **Описание:** Блок с контактами, статистикой и соцсетями
- **Элементы:**
  - `.infoGrid` - сетка карточек
  - `.infoCard` - карточка информации
  - `.infoCardTitle` - заголовок карточки
  - `.infoCardList` - список элементов

### 7. **Footer** (Подвал)
- **Файл:** `app/components/Footer.tsx`
- **Стили:** `app/components/Footer.module.css`
- **Описание:** Нижняя часть сайта с дополнительной информацией
- **Элементы:** Копирайт, ссылки, контакты

## Файлы стилей для разных макетов

### Стандартный макет (с боковой панелью)
- **Файл:** `app/page.module.css`
- **Описание:** Классический дизайн с кофейной палитрой

### Современный тёмный макет
- **Файл:** `app/components/page-modern.module.css`
- **Описание:** Тёмно-синий современный дизайн
- **Особенности:**
  - `.whySection` - секция "Почему выбирают нас"
  - `.missionSection` - секция "Наши направления"

### Тёплый уют (Bakery)
- **Файл:** `app/components/page-bakery.module.css`
- **Описание:** Тёплая бежево-коричневая палитра
- **Особенности:**
  - `.whySection` - секция "Почему выбирают СТЛ"
  - `.visitSection` - секция "Присоединяйтесь"

### Минималистичный макет
- **Файл:** `app/components/page-minimalist.module.css`
- **Описание:** Чёрно-белый с оливковыми акцентами
- **Особенности:**
  - `.productsSection` - секция специализаций
  - `.featuredSection` - избранная секция
  - `.statsSection` - статистика

## Как вносить изменения

1. **Изменить цвета:** Откройте нужный `.module.css` файл и измените значения цветов
2. **Изменить размеры:** Найдите нужный класс и измените `padding`, `margin`, `font-size`
3. **Изменить текст:** Откройте `.tsx` файл компонента и измените содержимое
4. **Добавить новую секцию:** Создайте новый блок в `.tsx` и стили в `.module.css`

## Визуальные помощники

На главной странице доступны:
- **Кнопка "📐 Показать структуру"** - открывает панель со структурой макета
- **Метки секций** - синие метки на каждой секции с названием и путём к стилям
- **Кнопка "Скрыть метки"** - скрывает метки секций
