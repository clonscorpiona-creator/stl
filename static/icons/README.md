# Наборы иконок для STL Platform

## Структура папок

```
static/icons/
├── default/      # Киберпанк-иконки (неоновые, с градиентами)
├── business/     # Деловые иконки (стильные, в цветовой палитре сайта)
└── minimal/      # Минималистичные иконки (тонкие линии)
```

## Доступные наборы

### Default (Cyberpunk)
- **Слаг:** `default`
- **Описание:** Высокотехнологичные киберпанк-иконки с неоновыми эффектами
- **Цвета:** Cyan (#00f5ff), Magenta (#ff00ff), Green (#00ff88)
- **Особенности:** Градиенты, glow-эффекты, анимация

### Business
- **Слаг:** `business`
- **Описание:** Стильные деловые иконки в цветовой палитре сайта
- **Цвета:** Primary (#5a3f8a)
- **Особенности:** Чистые линии, профессиональный вид

### Minimal
- **Слаг:** `minimal`
- **Описание:** Минималистичные иконки с тонкими линиями
- **Цвета:** Primary (#5a3f8a)
- **Особенности:** Легкость, простота, толщина stroke 1.5px

## Переключение наборов

Переключение между наборами иконок осуществляется через **Админ-панель Django**:

1. Войдите в админ-панель (`/admin/`)
2. Перейдите в раздел **"Наборы иконок"** (Core → IconSet)
3. Выберите нужный набор и установите флаг **"Активный набор"**
4. Сохраните изменения

Только один набор может быть активным одновременно. При активации нового набора, предыдущий автоматически деактивируется.

## Использование в шаблонах

Иконки автоматически подключаются через контекстный процессор `icon_set_context`:

```html
<!-- В <head> -->
<link rel="stylesheet" href="/static/icons/{{ icon_set }}/icons.css">

<!-- В навигации -->
<img src="/static/icons/{{ icon_set }}/home.svg" alt="" class="nav-icon">
```

## CSS классы

Каждый набор иконок включает CSS-файл со следующими классами:

- `.icon` — базовый размер (20px)
- `.icon-sm` — маленький размер (16px)
- `.icon-lg` — большой размер (24px)
- `.icon-xl` — XL размер (32px)
- `.icon-btn` — кнопка с иконкой
- `.nav-icon` — иконка в навигации
- `.icon-inverted` — инвертированная иконка (для темных кнопок)

## Добавление нового набора

1. Создайте папку с именем набора: `static/icons/your-set/`
2. Добавьте SVG-иконки с именами из списка выше
3. Создайте файл `icons.css` со стилями
4. В админ-панели создайте новую запись IconSet с нужным slug
