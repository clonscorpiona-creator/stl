───────────────────────────────────────────────────────────

🎯 РЕШЕНИЕ 1: Использовать существующую БД
───────────────────────────────────────────────────────────

Если существующая БД не используется или не важна:

1. Dashboard → Databases
2. Найдите существующую PostgreSQL БД
3. Откройте её
4. Вкладка "Connect"
5. Скопируйте "Internal Database URL"
6. Используйте этот URL для нашего проекта

Переходите к созданию Web Service (Шаг 2)

───────────────────────────────────────────────────────────

🎯 РЕШЕНИЕ 2: Удалить старую БД
───────────────────────────────────────────────────────────

Если старая БД не нужна:

1. Dashboard → Databases
2. Найдите старую PostgreSQL БД
3. Откройте её
4. Settings → Delete Database
5. Подтвердите удаление
6. Создайте новую БД "stl-db"

───────────────────────────────────────────────────────────

🎯 РЕШЕНИЕ 3: Upgrade на Starter Plan
