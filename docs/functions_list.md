# Список функций сайта STL

> Этот документ содержит все основные функции сайта STL. При добавлении новой функции:
> 1. Если функция запрошена пользователем - спросить название
> 2. Если функция системная - назвать самостоятельно в формате: `<app>_<action>_<object>`
> 3. Добавить запись в соответствующий раздел этого документа
> 4. Указать URL/Name и краткое описание

## Пользователи и аутентификация

| Функция | URL/Name | Описание |
|---------|----------|----------|
| Регистрация пользователя | `accounts:register` | Регистрация нового пользователя. Первый пользователь (ID=1) становится администратором |
| Вход пользователя | `accounts:login` | Аутентификация пользователя по username и паролю |
| Выход пользователя | `accounts:logout` | Завершение сеанса пользователя |
| Профиль пользователя | `accounts:profile` | Страница профиля с работами, коллекциями и статистикой |
| Подписка на пользователя | `accounts:follow_toggle` | Подписка/отписка от других пользователей |
| Редактирование профиля | `accounts:edit_profile` | Редактирование профиля: avatar, bio, website, location |

## Работы

| Удаление работы | `core:work_delete` | Удаление работы |

| Рекомендация администрации | `api:toggle_admin_recommend` | Staff может рекомендовать работу (бейдж "ТОП") |

## Индикаторы статуса пользователей

| Индикатор | Описание | Где отображается |
|-----------|----------|------------------|
| 🟢 Зеленый кружок | Пользователь активен, предупреждений нет | Работы, комментарии, чат |
| 🟡 Желтые кружки (1-3) | Количество предупреждений у пользователя | Работы, комментарии, чат |
| 🔴 Красный бейдж "БАН"/кружок | Пользователь забанен (автоматически при 4 предупреждениях или вручную) | Работы, комментарии, чат |
| 👑 ADMIN | Администратор сайта | Чат |
| 🛡️ MOD | Модератор чата/канала | Чат |

## Применение системы предупреждений

| Раздел | Функционал |
|--------|------------|
| Работы | Индикаторы у автора работы, кнопка "Предупреждение автору" |
| Комментарии | Индикаторы у авторов комментариев и ответов |
| Чат | Индикаторы у авторов сообщений, кнопка "Предупреждение" для staff |

## Автоматические функции

| Функция | Описание |
|---------|----------|
| Автобан при 4 предупреждениях | При получении 4-го предупреждения пользователь автоматически банится (срабатывает в модели Warning) |
| Подсчет статистики | Автоматический подсчет лайков, просмотров, комментариев, репостов, сохранений |
| Обновление счетчика предупреждений | При добавлении/удалении предупреждения счетчик обновляется автоматически в модели Profile |
| Сериализация сообщений чата | При отправке сообщения в чат автоматически добавляется информация о warning_count и is_banned пользователя |
